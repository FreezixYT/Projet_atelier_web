Git - Introduction

Git est un système de contrôle de version distribué qui permet de suivre les changements dans le code source pendant le développement d'un projet logiciel. Il facilite la collaboration entre les membres de l'équipe et permet de revenir à des versions antérieures du code.

Voici quelques concepts de base en Git :

COMMANDES ESSENTIELLES

Cloner un dépôt existant

            git clone <url-du-dépôt>

Initialiser un nouveau dépôt

            git init

Ajouter des fichiers pour le suivi

            git add <nom-du-fichier>

Enregistrer les modifications dans le dépôt

            git commit -m "Message de commit"

Vérifier l'état du dépôt

            git status

Envoyer les fichiers

            git push

CRÉER UN PROJET AVEC GIT

Créer un nouveau répertoire

            mkdir mon-projet

Initialiser un nouveau dépôt

            git init

Ajouter des fichiers au dépôt

            git add .

Enregistrer les modifications

            git commit -m "Initialiser le projet"
