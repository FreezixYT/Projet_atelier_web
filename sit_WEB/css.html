CSS (Cascading Style Sheets)

CSS est un langage de feuilles de style qui complète HTML en gérant la présentation et la mise en forme des pages web. Il permet de créer des designs attrayants en définissant les styles des éléments HTML, comme les couleurs, les polices, et les marges.

Voici quelques éléments de base en CSS :

SÉLECTION D'ÉLÉMENTS

            h1 {
                color: blue;
                font-size: 24px;
            }

            p {
                color: green;
            }

CLASSES ET ID

            .titre-important {
                font-weight: bold;
            }

            #conteneur {
                width: 80%;
                margin: 0 auto;
            }

BORDURES ET ARRIÈRE-PLANS

            .encadre {
                border: 1px solid #ccc;
                padding: 10px;
                background-color: #f0f0f0;
            }
